# CI/CD Project

## Description

This is a Node.js application used to demonstrate a complete DevOps CI/CD pipeline.

## Technologies Used

- GitHub
- Jenkins
- Docker
- Amazon ECR
- Kubernetes
- Prometheus
- Grafana

## Run Locally

Install dependencies:

```bash
npm install
```

Run the application:

```bash
npm start
```

Open your browser:

```
http://localhost:3000
```
